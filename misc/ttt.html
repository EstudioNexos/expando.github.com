Introduction

Easily initiate triggers to your (or someone else's) IFTTT recipes using your voice.

First join the Maker Channel on the IFTTT website. Then copy your private key to the space below.

Then create some recipes that use maker as the trigger and give each maker a unique name.

Then add the unique name to the triggers below.

Each trigger supports up to 3 optional values. Each value will be sliced from your speech based on word count.

Examples

  1. Turn on the porch lights
    Trigger Name = Turn on lights
    Use key = Default
    Value 1 = 1 word

    Say: Turn on lights porch

  2. Tell Mary that I will be late
    Trigger Name = Message
    Use key = Spoken
    Value 1 = Unlimited

    Say: Message Mary I will be late

  3. Email Bill with the subject memo and body you are great
    Trigger Name = Email
    Use key = Default
    Value 1 = Dictionary
    Value 2 = 1 word
    Value 3 = Unlimited

    Say: Email Bill memo you are great
